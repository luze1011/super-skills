### TDD 任务模板

```markdown
### Task N: [功能名称]

**Files:**
- Test: `tests/[测试文件]`
- Code: `src/[源文件]`

**TDD 流程:**

- [ ] **RED: Write failing test**
  ```[语言]
  // 测试代码
  ```
  - 运行: `[测试命令]`
  - 预期: 测试失败

- [ ] **GREEN: Write minimal code**
  ```[语言]
  // 最小实现代码
  ```
  - 运行: `[测试命令]`
  - 预期: 测试通过

- [ ] **REFACTOR: Improve code**
  - 检查代码质量
  - 运行: `[测试命令]`
  - 预期: 测试仍然通过

- [ ] **COMMIT**
  ```bash
  git add .
  git commit -m "feat: [功能描述]"
  ```
```
