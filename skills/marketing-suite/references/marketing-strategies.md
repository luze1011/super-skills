# 营销策略参考手册

## 一、经典营销模型

### 1. AIDA 模型
**注意(Attention) → 兴趣(Interest) → 欲望(Desire) → 行动(Action)**

适用场景：广告文案、销售页面
- **注意**：用标题、视觉抓住用户眼球
- **兴趣**：提供有价值的信息引发好奇
- **欲望**：展示产品价值激发购买欲
- **行动**：明确的CTA引导转化

### 2. PAS 公式
**问题(Problem) → 煽动(Agitate) → 解决方案(Solution)**

适用场景：痛点营销、产品推广
- **问题**：指出用户面临的困扰
- **煽动**：放大痛点，让用户感同身受
- **解决方案**：引出产品作为最佳答案

### 3. FAB 法则
**特性(Feature) → 优势(Advantage) → 利益(Benefit)**

适用场景：产品介绍、卖点提炼
- **特性**：产品有什么功能
- **优势**：比竞品好在哪里
- **利益**：给用户带来什么好处

## 二、平台运营策略

### 小红书
| 要素 | 最佳实践 |
|------|---------|
| 标题 | 数字+痛点+好奇心，如"3个方法让你..." |
| 封面 | 高颜值、对比图、文字点缀 |
| 正文 | 真实体验+干货输出+emoji丰富 |
| 标签 | 5个左右，包含热门词和长尾词 |
| 发布 | 18:00-22:00黄金时段 |

**爆款公式**：
- 种草笔记 = 痛点引入 + 真实体验 + 效果对比 + 购买引导
- 测评笔记 = 开箱 + 使用感受 + 优缺点总结 + 购买建议

### 抖音
| 要素 | 最佳实践 |
|------|---------|
| 开头 | 3秒黄金法则，用悬念/冲突/提问抓住注意力 |
| 节奏 | 短句、快切、信息密度高 |
| 结尾 | 引导互动（点赞/评论/关注）|
| BGM | 蹭热点音乐，节奏卡点 |

**爆款公式**：
- 剧情类 = 冲突开场 + 反转 + 情感升华
- 口播类 = 痛点提问 + 解决方案 + 行动号召
- 种草类 = 震惊开场 + 产品演示 + 限时优惠

### 微信公众号
| 要素 | 最佳实践 |
|------|---------|
| 标题 | 好奇心/利益点/数字/名人效应 |
| 开头 | 用故事/数据/痛点引入 |
| 结构 | 小标题分段，每段不超过300字 |
| 结尾 | 总结+互动+引导关注 |

## 三、内容营销框架

### 1. 内容矩阵
```
                    高价值
                      │
    深度内容(长文)    │    系列内容(专题)
                      │
──────────────────────┼──────────────────────
                      │
    日常内容(短文)    │    互动内容(话题)
                      │
                    高频次
```

### 2. 内容日历规划
| 类型 | 频率 | 内容方向 |
|------|------|---------|
| 日常 | 每天 | 轻量级内容，维持活跃 |
| 干货 | 每周2次 | 专业内容，建立权威 |
| 活动 | 每月1次 | 营销活动，促进转化 |
| 热点 | 随时 | 追热点，借势传播 |

## 四、用户运营策略

### RFM 模型
- **R(Recency)**：最近一次消费时间
- **F(Frequency)**：消费频率
- **M(Monetary)**：消费金额

**用户分层运营**：
| 用户类型 | 特征 | 运营策略 |
|---------|------|---------|
| 重要价值 | R↑F↑M↑ | VIP服务，专属权益 |
| 重要发展 | R↑F↓M↑ | 提升消费频次 |
| 重要保持 | R↓F↑M↑ | 召回激活 |
| 一般价值 | R↑F↑M↓ | 提升客单价 |

## 五、数据分析指标

### 核心指标体系
```
曝光层：曝光量、CPM、到达率
├── 互动层：点赞、评论、转发、互动率
│   ├── 转化层：点击率、转化率、CPC、CPA
│   │   └── 收入层：GMV、ROI、客单价、复购率
```

### 各阶段关注指标
| 阶段 | 核心指标 | 目标 |
|------|---------|------|
| 认知 | 曝光量、触达人数 | 让更多人知道 |
| 兴趣 | 点击率、互动率 | 让用户感兴趣 |
| 考虑 | 留资率、加购率 | 让用户深入了解 |
| 转化 | 转化率、GMV | 完成购买 |
| 忠诚 | 复购率、NPS | 成为忠实用户 |
