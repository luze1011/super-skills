# 营销套件

一站式营销内容创作与活动策划工具集。

## 核心能力

| 模块 | 功能 | 输出 |
|------|------|------|
| 内容创作 | 生成营销文案、社媒内容 | 多平台适配文案 |
| 活动策划 | 制定营销活动方案 | 完整策划文档 |
| 内容日历 | 规划内容发布计划 | 排期表格 |

## 使用流程

### 1. 内容创作

```bash
python scripts/content-create.py --platform <平台> --topic <主题> --style <风格>
```

支持平台：
- `xiaohongshu` - 小红书（种草笔记、测评分享）
- `douyin` - 抖音（短视频脚本）
- `weibo` - 微博（话题文案）
- `wechat` - 微信公众号（长图文）
- `all` - 全平台适配

### 2. 活动策划

```bash
python scripts/campaign-plan.py --type <类型> --budget <预算> --duration <周期>
```

活动类型：
- `brand` - 品牌活动
- `promotion` - 促销活动
- `launch` - 新品发布
- `festival` - 节日营销

### 3. 参考资源

- **营销策略库**: `references/marketing-strategies.md` - 营销方法论与案例
- **内容模板库**: `references/content-templates.md` - 各平台内容模板

## 输出格式

### 营销文案结构
```markdown
## 标题（吸引眼球）
[正文内容]

### 核心卖点
- 卖点1
- 卖点2

### 行动号召
[CTA内容]

### 话题标签
#标签1 #标签2
```

### 活动策划结构
```markdown
# [活动名称]

## 活动背景
## 活动目标
## 目标人群
## 活动内容
## 传播策略
## 预算分配
## 时间规划
## 风险预案
## 效果评估
```

## 最佳实践

1. **平台适配** - 不同平台有不同的内容风格，注意调整
2. **数据驱动** - 参考历史数据优化内容
3. **测试迭代** - 小范围测试后放量推广
4. **合规审查** - 确保内容符合平台规则
